Dreamlink Quark Player 说明文档
------------------------------

Dreamlink Quark Player 是运行在服务器端的播放器程序，主要实现功能如下

- 网络广播输出
- 声卡输出
- 标准输入输出通讯
- 支持通过管道输入外部音频数据


## 命令参数定义

### --output 输出模式
***缩写 -o***

输出模式有两种，分别是通过网络输出和通过通过本地声卡输出，对应参数分别是，默认的输出方式是网络输出(net)

**--output net** 网络输出

**--output local** 本地声卡输出
